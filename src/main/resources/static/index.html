Calendar API

WARNING: THIS API USES IN-MEMORY DATABASE. ALL DATA IS LOST WHEN APPLICATION IS SHUTDOWN!

Usage:

/events (GET) - Get all stored events

/events (POST) Save new event as JSON. Fields:

  • name - Text
  • length - Numeric
  • date - (Format: yyyy.MM.dd HH:mm)

Additional Search features for /events (GET). Used as query parameters. All parameters can be used individually or as a combination

  • ?year
  • ?month
  • ?day
  • ?time (Format: HH:mm)
/eventsbyname?name (GET) finds all events by a name

/eventsongoing (GET) finds all ongoing events
